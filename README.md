# Jobvite

Fetch jobs from Jobvite.

## Installation

Add this line to your application's Gemfile:

    gem 'jobvite'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jobvite

## Usage

```ruby
jobvite = Jobvite.new('COMPANY_ID')
jobvite.jobs
```
